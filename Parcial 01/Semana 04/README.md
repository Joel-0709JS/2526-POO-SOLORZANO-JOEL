Ejemplos del Mundo Real con Programación Orientada a Objetos (POO)

Este repositorio contiene dos ejemplos prácticos que aplican los principios de la Programación Orientada a Objetos (POO) en Python. Cada ejemplo modela un sistema del mundo real, con clases bien definidas, encapsulamiento, interacción entre objetos y diseño modular.



&nbsp;Ejemplo 1: Tienda de Ropa — tienda\_ropa.py

Descripción

Simula una tienda de ropa llamada "Joel J. S Express", donde los clientes pueden comprar productos como camisas, pantalones y zapatos...



Principios de POO aplicados

Encapsulamiento: Atributos privados (\_\_nombre, \_\_precio, \_\_stock)...

Herencia: Las clases Camisa, Pantalon y Zapato heredan...

Polimorfismo: Cada tipo de prenda redefine el método mostrar\_info()...
